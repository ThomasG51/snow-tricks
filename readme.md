###### How to install ?
    git clone https://github.com/ThomasG51/snow-tricks.git

    cd snow-tricks

configure .env file with the database credential

    composer install

    yarn install

    doctrine:database:create

    doctrine:migrations:migrate

    symfony server:start

###### Frameworks/Libraries
* Symfony
* Tailwind.css
